# Ubots Messages Server

## Before all

Create a *.env* file and set: **DATABASE_URL** and **DATABASE_AUTH_TOKEN**. Here is using Turso (SQlite) with Drizzle ORM.

Ex:
```bash
DATABASE_URL=libsql:${DATABASE_URL}
DATABASE_AUTH_TOKEN=${DATABASE_AUTH_TOKEN}
```
 
## Install

```bash
pnpm i
```

## Install

```bash
pnpm prepare
```

## Running

```bash
pnpm dev
```
